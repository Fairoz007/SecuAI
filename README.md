# SecuAI

A web application developed by Fairoz Faisal.

## About

**SecuAI** is a web application that analyzes and identifies vulnerabilities in smart contracts. It helps developers to secure their smart contracts by providing a detailed report of the vulnerabilities found.

## Features

*   **Smart Contract Analysis:** Analyze smart contracts for vulnerabilities.
*   **Vulnerability Reports:** Generate detailed reports of the vulnerabilities found.
*   **Secure Coding Guidelines:** Provides guidelines for writing secure smart contracts.

## Technologies Used

*   **Frontend:**
    *   Next.js
    *   React
    *   TypeScript
    *   Tailwind CSS
    *   Shadcn UI
*   **Backend:**
    *   Node.js
    *   Supabase

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js
*   npm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/SecuAI.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Create a `.env.local` file and add your Supabase credentials.
    ```
    NEXT_PUBLIC_SUPABASE_URL=YOUR_SUPABASE_URL
    NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
    ```

### Usage

To start the development server, run:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
